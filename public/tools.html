HERRAMIENTAS SOPORTE A SCRUM

A continuación se ofrece una serie de herramientas para dar soporte a los diferentes componentes de Scrum. Entre dichos componentes se encuentran las Daily Meetings, las priorizaciones, las estimaciones y las retrospectivas. Además, en el caso de las priorizaciones y las estimaciones, encontramos diferentes técncias que podemos aplicar para llevarlas a cabo.

En el caso de las priorizaciones, podremos elegir entre las siguientes técnicas:

  • Dot Voting
  • MoSCoW priorization

    • En el caso de las estimaciones, podremos elegir entre las siguientes técnicas

  • Poker Game
  • T-shirt
  • Cálculo de velocidad

HERRAMIENTAS PARA SOPORTE DE EVENTOS

A continuación se ofrece una serie de herramientas para dar soporte a las diferentes técnicas usadas en las Metodologías Ágiles. Entre dichas técncias o procedimientos, se encuentran las daily meetings, las priorizaciones, las estimaciones y las retrospectivas. Además, en el caso de las priorizaciones y las estimaciones, encontramos diferentes técncias específicas que podemos aplicar para llevarlas a cabo.

En el caso de las priorizaciones, podremos elegir entre las siguientes técnicas:

  • Dot Voting
  • MoSCoW priorization

    • En el caso de las estimaciones, podremos elegir entre las siguientes técnicas

  • Poker Game
  • T-shirt
  • Cálculo de velocidad
